Firebase conectado ✅
Sincronização em tempo real ativa
1
Acesse console.firebase.google.com e faça login com sua conta Google
2
Clique em Criar projeto → nome: gaara-eren → desativar Google Analytics → Criar
3
No menu lateral: Criação → Realtime Database → Criar banco de dados → Modo de teste → Criar
4
Vá em Visão geral do projeto ⚙️ → Configurações → Seus aplicativos → clique no ícone Web </> → registre o app → copie o objeto firebaseConfig
5
Abra este arquivo HTML em qualquer editor de texto (Bloco de Notas, VS Code) → procure por FB_CONFIG → substitua cada COLE_AQUI_xxx pelo valor correspondente do seu firebaseConfig
6
Salve o HTML e publique no GitHub Pages (ou Netlify Drop). Mande o link pro seu sogro. Pronto — qualquer edição que você fizer aparece na hora no celular dele! ✅
